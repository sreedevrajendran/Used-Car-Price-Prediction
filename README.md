# Used-Car-Price-Prediction
This project builds a linear regression model to predict used car prices. It covers data loading, cleaning (handling missing values, type conversion), feature engineering, model training, prediction, and evaluation, including visualizations for insights into car data distributions.

 It involves several key steps:

Data Loading and Initial Exploration: Loading the car dataset and performing initial checks on its structure and basic statistics.
Data Cleaning and Preprocessing: Handling missing values, converting data types for various features like 'Mileage', 'Engine', 'Power', and 'New_Price', and imputing missing 'Seats' values.
Feature Engineering and Data Splitting: Preparing the data for model training by dropping irrelevant columns and splitting the dataset into training and testing sets.
Model Training: Training a Linear Regression model on the prepared training data.
Prediction and Evaluation: Using the trained model to make predictions on new car data and evaluating its performance using the R-squared score.
Data Visualization: Exploring relationships between key features like 'Kilometers_Driven' and 'Price', and analyzing the distribution of car prices by 'Fuel_Type' to gain insights into the dataset.
